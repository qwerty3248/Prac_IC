Para salir luego de haber puesto clips en la terminal, escribe: (exit)

Escribe antes de ejecutar el programa: (load "mi_programa.clp") para cargarlo
Luego haz un (list) para ver los programas cargados, en verdad no funciona hay que listar cada lista y cada hecho
--Luego haz un (clear all) para borrar los programas cargados si quieres borrarlos 
--Luego haz un (reset) para borrar los programas cargados y los datos de la base de datos
Para ejecutar el programa, escribe: (run)

La extension ".clp" es para programas en CLIPS
